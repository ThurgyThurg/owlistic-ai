## Features

- 📒 Notebooks/Notes tree
- ✏️ Rich (WYSIWYG) editor
- ✔️ Inline todo items
- 🔄 Real-time sync
- 🔑 JWT-based auth
- 🔒 Role-based access control
- 🗑 Trash
- 🌓 Dark/Light mode
- ⬇️ Import markdown notes
