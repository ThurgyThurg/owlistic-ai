# Repository Structure


## Directory Structure

The repository is organized into several directories to facilitate code organization and maintainability. Here's a breakdown of the main directories:

### Server

```text
src/backend/
├── cmd/main.go
├── config
├── database
├── models
├── middleware
├── routes
├── services
├── broker
└── utils
```

### App

```text
src/frontend/lib
├── main.dart
├── core
├── models
├── providers
├── screens
├── services
├── viewmodel
├── widgets
└── utils

```
